WORKING WITH GIT

Making a new repository

Sign up/sign into GitHub, download GitBash and authorise...

On GitHub, create a new repository.

Follow the instructions on the website GitHub.

When creating new repository, you can type 'git add .' to add all files to your repository that are in your VScode project ('.' = all).

Commands:

git init
git add . ("." adds all files)
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/crimscode/workingWithGit.git (link is your repository)
git push -u origin [name of branch to default]
